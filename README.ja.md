# My Ideals - 生写真コレクション記録アプリ

[English](./README.en.md) | [日本語](./README.ja.md) | [中文](./README.zh.md)

My Ideals は、アイドルの生写真コレクションを管理するための Web アプリです。

- ✅ 持っている写真をチェック
- 📊 コレクション進捗を一目で確認
- 🔍 素早く検索・フィルター
- 💾 データはブラウザに保存 - アカウント不要

## クイックスタート

### ⚠️ 注意

本アプリのすべてのデータはブラウザのローカルにのみ保存されます。ブラウザデータの消去、シークレットモードの使用、デバイスの変更により、データは永久に失われます。定期的にバックアップをエクスポートしてください。**開発者はデータ損失による損害について責任を負わず、失われたデータの復旧もできません。**

https://my-ideals.notequal.me/

### プロファイルを作成

1. 以下のメニューから **「新規プロファイル」** を選択します。
   - **PCの場合**：右上のドロップダウンリストをクリック
   - **スマホの場合**：右上の三本線（ハンバーガーメニュー）をクリック
2. テンプレート URL を貼り付け（コミュニティやテンプレート作成者から取得）
3. プロファイル名を入力
4. **「作成」** をクリック

### メンバーでフィルター

- メンバーフィルターボタンで特定のメンバーのみ表示
- 複数メンバーを含むテンプレートで便利

### コレクションを記録

1. コレクションを閲覧
2. アイテムをクリックして **所持** ✓ をマーク
3. もう一度クリックで解除
4. 進捗は自動保存されます

### データをバックアップ

- 右上の **「エクスポート」** をクリックしてデータファイルをダウンロード
- 大切に保管し、必要に応じてインポートして復元できます

## 仕組み

**テンプレート (Template)** = コミュニティが作成した写真リスト

テンプレートは、セットにどんな写真があるかを定義します（例：「2024年夏コンサートグッズ」）。テンプレート作成者がリストをメンテナンスするので、自分で一つ一つ入力する必要はありません。テンプレートURLを貼り付けるだけで始められます。

**プロファイル (Profile)** = あなた個人のコレクション記録

プロファイルはテンプレートを使って、どのアイテムを持っているかを記録します。複数のプロファイルを持つことができます（例：グループごと、トレード用など）。

```
テンプレート（共有）     プロファイル（自分用）
┌─────────┐            ┌─────────┐
│ A       │            │ A    ✓  │  ← 所持
│ B       │    -->     │ B       │
│ C       │            │ C    ✓  │  ← 所持
│ ...     │            │ ...     │
└─────────┘            └─────────┘
```

## よくある質問

**Q: データはどこに保存されますか？**

A: すべてのデータはブラウザのローカルストレージに保存されます。サーバーには送信されません。

**Q: ブラウザデータを消去したら？**

A: コレクションデータは失われます。消去前に必ずバックアップをエクスポートしてください。

**Q: 複数のデバイスで使えますか？**

A: はい。ただし自動同期はありません。一方のデバイスからエクスポートして、もう一方でインポートしてください。

**Q: テンプレートとは何ですか？**

A: テンプレートは写真リストを定義する JSON ファイルです。テンプレート作成者がリストをメンテナンスするので、ユーザーは手動で入力する必要がありません。

**Q: テンプレート URL が動作しません**

A: **「テンプレートURLを編集」** をクリックしてリンクを更新してください。テンプレートが移動した場合は、作成者から新しい URL を取得してください。
